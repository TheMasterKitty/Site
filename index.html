RNGMAX

RNGMAX is a bot that can do many things related to random numbers including a coin flip, 8-ball, and rolling dice! It has plenty of random number generation commands to spice up your Discord Server for free (of course)! To check the command list at any time, run /help as there are often new commands. If you want a new command added, type /suggestion <command idea> and it is likely that suggestion will be added to RNGMAX

Privacy Policy:
RNGMAX stores no data from members, users, or owners that is not necessary. Necessary data includes: the guild id* for storing /server-settings (only if applicable) and the user id* for storing /user-settings (only if applicable). It also stores the last command ran (across the entire bot) for bug finding and the amount of a specific command ran for statistics (neither of these contain even a user id of who ran the command). The bot does not store or log anything else and all data is privately stored besides the top command bot-wide for the bot's status.

* Guild IDs are a public identifier for Discord Servers and is public to all users in a server
* User IDs are a public identifier for any Discord User and can be viewed by anyone across Discord

Ruby

Ruby is a bot focused around using Discord's built-in features instead of adding excessive commands. It allows for logging and DMing users when they are punished via Discord's built in timeouts, kicks, bans, etc. It only requires audit log permissions to view information on who punished the user.

Privacy Policy:
Ruby stores all log messages from your server along with the following information: your server id (public id/identifier for your server), the user id (public id/identifier for any discord user), and the exact timestamp of the action. These logs can be cleared by using the command /clearlogs <user> or by kicking Ruby from your server and will not be able to be recovered. These logs are nessecary and are not shared with anyone besides you and anyone with Timeout Members permissions on your server running the /logs <user> command.

Invite Cleaner

Invite Cleaner is a very simple bot that automatically clears old invites to your Discord Server (as there is a limit). You can set it up with the command /autodelete <date> where the date is how many days an invite can stay before being deleted. From there, you can sit back as any old discord invites are removed!

Privacy Policy:
Invite Cleaner has absolutely no reason to store any data and therefore doesn't. The only thing that is saved is your public Guild Identifier and the date you set via /autodelete <date>.
